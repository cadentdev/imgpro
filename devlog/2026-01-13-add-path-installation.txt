## Notes

The wrapper script approach was chosen because:
- imgpro depends on Pillow and pillow-heif which are installed in a venv
- A simple symlink would use system Python which may not have these dependencies
- The wrapper script explicitly uses the venv's Python interpreter

Wrapper script location: `/usr/local/bin/imgpro`
Wrapper script content:
```bash
#!/bin/bash
/path/to/imgpro/.venv/bin/python /path/to/imgpro/imgpro.py "$@"
```
